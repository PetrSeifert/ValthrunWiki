## What is Valthrun
Valthrun is a suite of tools developed to enhance gaming experiences by providing applications that offer additional game insights and functionalities.
These tools range in purpose, such as displaying in-game overlays or broadcasting radar information independently.
The heart of Valthrun are hereby the drivers which provide critical functionalities but remain completely transparent to the Windows operating system.

:::warning
Valthrun is **not** a plug 'n play solution.  
We aim to achieve maximum stealth and stay transparent for other applications.  
Please read [getting started](./getting-started/) carefully and try to troubleshooting issues on your own.  
:::

## Supported Games

Currently Valthrun focuses mainly on Counter Strike 2 but support for other games are in development.  

### Counter Strike 2 enhancements
- an ingame overlay for
  - Player ESP  
  - Trigger Bot  
  - Spectator info  
  - Bomb Info
  - Stream proof by default
  - ... and more  
- a web based radar

### PUBG enhancements
- a terminal based enhancer providing information about enemy players
